# Moye
game framework of cocos creator & nodejs  

this repository is Moye Client, support cocos creator version 3.8^

### install to cocos creator project
```bash
npm i github:Alistairot/moye-client
```

### how to compile src?
```
npm install
cd gulp
gulp build
```
